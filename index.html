Future Planning Idea

Portable Designer

Overview/Content

Many architecture designs on computers. But what if they have to travel. I was having this idea as a tape measure, where the ruler can be pulled out so when connecting to the technology, we can design it into when pressing a button, the screen will be able to be move /go back. Architecture is also designing and making blueprints on paper, but if they lose it and use too much paper, it is bad for the environment. But what happens if they need to travel, they won’t be able to bring a whole computer with them. So let's see my idea.

  • So I was thinking, if we can make a portable and foldable/roll up technology, it would be much easier and design everywhere they want.
  • They can open/pull out an electronic scroll and start designing and pull the pen out of the side corner.
  • It will be waterproof, so the architecture will be able to be not worried about the design getting wet like how in paper
